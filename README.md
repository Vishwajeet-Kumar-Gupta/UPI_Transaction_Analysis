# UPI_Transaction_Analysis
Project Overview

This project presents an end-to-end analysis of a UPI (Unified Payments Interface) transaction dataset using Excel, Python, SQL, and Power BI. The objective is to validate data quality, analyze transaction behavior, detect fraud patterns, and build interactive dashboards to support data-driven decision-making.

The project covers the complete analytics workflow, including data validation, exploratory data analysis (EDA), SQL querying, statistical analysis, and business intelligence visualization.

🎯 Objectives
Validate data quality and referential integrity.
Analyze customer and merchant transaction behavior.
Detect fraud patterns and transaction anomalies.
Identify business trends using statistical analysis.
Build interactive dashboards for business insights.
